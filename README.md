# motor-quantitativo-alm
Motor quantitativo (Python) para Gestão de Tesouraria (ALM). Automatiza o risco institucional unindo o rigor da FIPECAFI a pipelines de dados. Arquitetura engloba: Ingestão de políticas (PDF/JSON), Construtor de Curva Spot B3 (Spline Cúbica) e Simulador de estresse LCR (Basileia). Construído no Lab de Eng. Financeira.
